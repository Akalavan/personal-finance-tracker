# Personal Finance Tracker

Микросервисная система для учета личных финансов и инвестиций.

## Модули

- Auth Service
- Finance Service
- Investment Service
- Report Service
- Notification Service
- API Gateway
- Config Server
- Discovery Server

## Технологии
- Java 21
- Spring Boot 3.2
- Gradle Kotlin DSL
- Docker, Docker Compose
- Kafka, PostgreSQL, Redis
- Prometheus, Grafana, Zipkin
